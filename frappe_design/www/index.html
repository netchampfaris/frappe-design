Frappe Design

Frappe Design is a collection of design resources, reusable components and guidelines for creating Frappe products.

Brand

Branding guidelines and assets including foundations like colors, typography, writing style and logos.

Components

UI components implemented in Vue as well as raw HTML with encapsulated styles and tailwind based theming.

Marketing

Layouts and building blocks in HTML used to build marketing pages for Frappe products.
